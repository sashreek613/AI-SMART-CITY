# AI Smart City Dashboard

An AI-powered smart city platform that allows citizens to report urban issues such as potholes, garbage accumulation, and traffic congestion using an interactive map interface.

## Hackathon Project
This project was developed during a hackathon to demonstrate how AI and data visualization can help cities identify problem hotspots and improve urban issue management.

## Features
- Interactive map for reporting city problems
- Automatic location detection
- Heatmap visualization of issues
- Dashboard for analyzing complaints
- Sentiment analysis of citizen reports

## Tech Stack
- Python
- Streamlit
- Folium
- Pandas
- TextBlob

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py
